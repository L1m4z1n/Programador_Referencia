HTML
Criar uma página simples
Crie uma página HTML com título, um parágrafo e uma imagem. Adicione um link para um site externo.

Criar uma lista de itens
Crie uma página com uma lista ordenada (numerada) e uma lista não ordenada (bullets), contendo ao menos 5 itens em cada lista.

Criar uma tabela simples
Crie uma tabela com 3 colunas e 4 linhas. Adicione dados fictícios e cabeçalhos para as colunas.

Formulário de contato básico
Crie um formulário com campos para nome, email e uma mensagem, com um botão para enviar.

CSS
Estilizar texto

Alterar fundo da página
Defina uma cor de fundo para a página, usando o código hexadecimal ou nome da cor.

Estilizar listas
Estilize uma lista não ordenada com bullets personalizados (exemplo: círculos, quadrados).

Alinhar texto
Alinhe um texto ao centro, à esquerda e à direita.

JavaScript

Alterar conteúdo de um elemento
Crie um script que altere o conteúdo de um parágrafo quando o usuário clicar em um botão.

Mostrar data e hora atuais
